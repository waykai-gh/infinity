Windows

Скачать клиент

Для Windows используйте приложение NekoBox:

1. Перейдите в официальный сайт NekoBox

2. Скачайте последнюю версию в разделе загрузок

Запустите NekoBox и перейдите во вкладку "Сервер"
Нажмите на "+" в правом верхнем углу
Выберите "Импорт из буфера обмена" или "Импорт из URL"
Вставьте предоставленную ссылку и нажмите "OK"
Выберите сервер из списка и нажмите "Запустить"
Импорт конфигурации через URL
