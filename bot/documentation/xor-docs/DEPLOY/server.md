# Документация сервера mySrv (ru-XOR)

## Информация о сервере

- **Хостинг-провайдер:** firstvds.ru
- **IPv4 адрес:** 62.181.44.4

## Подключение

Подключение выполняется по SSH с публичным ключом на сервере (`authorized_keys`) и приватным ключом на ПК. Подключение также возможно без пары ключей, по паролю админа.

## Установленное ПО

На сервере установлены Docker и Git для управления версиями и образами, в общем проектом, для деплоя с локального ПК на сервер.

## Расположение проекта

Папка самого проекта — `infinity` содержит все нужные файлы проекта для запуска.
